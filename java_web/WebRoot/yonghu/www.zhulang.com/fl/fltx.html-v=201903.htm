2017逐浪小说作者福利计划

福利说明

福利的调整是为了让大家更安心创作，向读者展现更完美的自己

（1）全新全勤制度：签约有全勤，买断也有全勤，新上架作品加倍发，再也不用担心吃土了。
（2）打赏分成提高：打赏分成提高至80%，发动读者们一起嗨起来吧，赏赏赏发发发。
（3）完本奖金增加：完本字数越多，奖金越多，最高可得一万元完本奖，意外惊喜是不是。
（4）健康计划升级：意外保额40万，大病保额20万，健康后顾无忧可以安心创作。
（5）本版实施时间：2017年7月1日—2017年12月31日。

1成神直通车

1. 实施对象

所有独家签约逐浪网的，包括电子分成和买断作品，表现杰出的作者优先考虑。

2. 实施细则

逐浪网接下来将加大作品在手游、影视、漫画、有声改编等方面的版权拓展，目前已经在原创漫画、有声小说、网络大电影方面都有作品，已上线的人气漫画作品包括《宛香》、《唐寅在异界》、《民国妖闻录》……等等。

3买断计划

1. 实施对象

（1）有读者资源的优秀作者。
（2）在逐浪网有vip完本作品的作家（优先）。
（3）他站有过百万字完本经验的优秀作者。

2. 实施细则

作品申请买断，视作品质量，提供多种买断方式：

（1）全本书以千字15元起价格买断，上不封顶。
（2）分成转买断，分成签约作品上架满30天后，可向责任编辑申请转普通买断。

3. 本站作者可向责任编辑申请，由责任编辑审核确认后，编辑部审核确定。
他站作者将申请作品发邮件至以下邮箱：tougao@zhulang.com ，或通过本页下方的编辑QQ投稿。

4保底分成计划

1. 实施对象

（1）读者资源的优秀作者。
（2）在逐浪网有vip完本作品的作家 （优先）。
（3）他站有过百万字完本经验的优秀作者。

2. 实施细则

投稿事项：
（1）投稿作品需附带一份完整大纲和三万字正文，以及作者的意向买断价格，以千字为单位，如税前千字20元。若在他站有过较好成绩，可截图随邮件发送。
（2）当作品通过后，会有相关编辑与作者联系。
保底细则：
（1）全本书以千字10元起，上不封顶，保底作品享受全勤详见全勤方案。
（2）保底作品稿费：以分成稿费和保底稿费两者高者发，总分成稿费超出总保底稿费的作品完本以后，仍继续按月发放分成稿酬。
（3）本计划参与过程中采用欺骗手段或违禁手段以达到该计划所规定的字数，经举报查实者网站有权不予发放买断稿酬或进行相关处理。
本站作者可向责任编辑申请，由责任编辑审核确认后，编辑部审核确定。他站作者将申请作品发邮件至以下邮箱：tougao@zhulang.com

3. 具体细则

（1）一名逐浪作家，如果有多部作品，同时上了鲜花榜，可以得到所有作品相对应的奖励。
（2）设置鲜花年榜
   a、鲜花年榜第1名奖励价值9999元的礼物
   b、鲜花年榜第2名奖励价值7888元的礼物
   c、鲜花年榜第3名奖励价值5888元的礼物
   d、鲜花年榜新人王奖励价值6666元的礼物
鲜花年榜第一名是指逐浪网所有签约作家中，其逐浪网上架作品，在该年度所得鲜花总和第一名，其他以此类推。鲜花年榜新人王指该年度在逐浪网新驻站的作家中，所有逐浪网上架作品，该年在逐浪网的鲜花总和第一名获得者。鲜花年榜第一名和新人王不重叠，如有新作者当年即获得鲜花年榜第一名，则鲜花年榜新人王由新作者第二名获得。
（3）老书只能获得鲜花月榜奖励，新书只能获得鲜花新书月榜奖励。
   a、新书：是指作品上架时间3个月内（包含上架当月）
        例如《神级修复高手》是2016年10月17日在逐浪网上架的，那么在2016年10月、2016年11月、2016年12月属于新书。
   b、老书：是指作品上架时间3个月后（包含上架当月）
        例如《神级修复高手》是2016年10月17日在逐浪网上架的，那么从2017年1月起，属于老书。
   c、上架时间以第一章VIP章节发布时间为准。
（4）新书在期限之内，可以多月重复获得鲜花新书月榜奖励。
（5）恶意作弊作品取消此计划资格。

8渠道增值分成

1. 实施对象

（1）逐浪网分成签约作品。
（2）逐浪网保底分成作品。

2. 执行方式

凡保底分成或分成签约的作品，均可获得逐浪小说网在全渠道收益的50%（扣除渠道运营成本）。

3. 结算及发放

（1）电信运营商渠道（俗称无线稿费）：按照惯例，本项福利将采取n＋6的发放模式（即当月产生的渠道分成收入，将会在6个月后，网站收到合作方渠道回款的当个结算月，随稿费以福利形式发放）。若合作方支付时间临时变更，请以编辑部通告为准。
（2）其他第三方渠道：根据合作方结算周期调整，按月发放，以实际发放为准。

4. 特别说明

逐浪网渠道资源丰富，目前已与WiFi万能钥匙、掌阅、QQ阅读、书旗小说、搜狗阅读……等等各大渠道展开合作。

9作家健康计划

1. 实施对象

无社保或商业保险的逐浪网全职作家

2. 内容方案

由逐浪网提供一份保额为40万元意外保险和一份保额为20万元的重大疾病保险。

3. 具体细则

符合条件的作者向责编发起申请，由主编审核通过后，每季度汇总投保。 在逐浪网创作过两本以上完本作品的作家优先参与。

11福利违规及其他说明

秉承对文学的梦想，为了扶持更多有志于从事于网文的作家，逐浪网推出了全新版的作家福利。而在福利执行的具体过程中，我们将常见问题总结如下。

1. 逐浪作家福利违规作品包括但不限于：

（1）内容抄袭。
（2）拼凑字数。
（3）假更新。
（4）恶意作弊，影响榜单公平公正。
（5）存在损害读者、其他作者及逐浪网的行为。
（6）协议期间内，在未告知逐浪的情况下中断创作转签外站。
（7）与逐浪网版权合作协议相关规定相违背。
（8）违反国家相关法律法规。

2. 福利取消细则及告知：

（1）福利取消分以下几种：
存在违规，但情节不严重，能配合网站及时弥补的，逐浪网将取消作品当月福利资格。 严重违反职业道德，损害读者，其他作者及逐浪利益，或者在违规之后，不配合网站及时弥补的，逐浪网将取消该作品所有的福利资格。
（2）违规的告知与查询
每一位存在违规情形的作者我们将会通过作家专区邮件、QQ、邮件等形式告知。或者，您也可以联系您的责编进行查询确认。

3. 其他说明：

（1）本版作家福利计划起始时间（2017年7月1日至2017年12月31日）
（2）逐浪网拥有本福利计划的最终解释权，并保留根据实际情况对本福利计划及具体操作方式进行修订的权利。
