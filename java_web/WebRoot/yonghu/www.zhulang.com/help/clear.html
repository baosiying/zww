你我携手共创清朗网文空间

----逐浪网致广大作者、读者的一封公开信

　

最亲爱的作者、读者朋友们：

　　逐浪网自2003年上线以来，在各位亲爱的作者与读者的努力与厚爱之下，已成为业内极具影响力的主流网络文学平台。随着用户量的增长和社会影响力的提升，我们也深深感觉到肩负的社会责任也越来越重大。

　　为了响应习近平总书记的号召，建立清朗的网络空间，同时贯彻国家新闻出版署、全国“扫黄打非”工作小组办公室《关于开展网络文学专项整治行动的通知》精神，更好地满足人民群众美好的精神文化生活需求，我们对逐浪网的作者提出以下倡议：

一、坚持正确的价值导向，自觉抵制创作违背社会主义核心价值观的低俗、庸俗、媚俗作品；

二、加强自净意识，坚决杜绝创作含有淫秽色情、血腥暴力、恶搞经典、抹黑英雄的网络文学作品；

三、保护未成年人，自觉做到作品中不出现校园暴力、校园霸凌、色情、低俗等不利于未成年人健康成长的内容；

四、坚持文化自信，努力创作具有中国特色、中国风格、中国气派，体现中华文化精髓，反映中国审美追求，表现中国人民情怀的作品；

五、坚守文学的理想与责任，弘扬真善美，发挥正能量，努力创作出更加优秀的网络文学作品，在浩瀚的中国文学殿堂中留下属于自己的印记。

目前，逐浪网已通过各种举措加大内容把关力度，同时，我们也真诚邀请各位忠实读者帮助我们，监督我们的工作。我们已在每个阅读页面开通举报反馈功能，读者在阅读时发现淫秽色情、血腥暴力、抹黑英雄等违规内容，均可点击页面上的举报按钮进行举报。我们收到举报后会第一时间对该内容进行核查处理，每个月会汇总举报信息，对有效举报违规内容的用户给予逐浪币奖励。；

　　让我们共同携手，创作并传播更多弘扬中国精神，凝聚中国力量的精品力作，一起将逐浪网打造成为向上向善、积极清朗的网络文学之海，让广大作者、读者尽情遨游逐浪！
